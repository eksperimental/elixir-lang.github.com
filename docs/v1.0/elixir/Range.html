    Elixir v1.0.5 Range

    Defines a Range.

    A Range is represented internally as a struct. However, the most common form of creating and matching on ranges is via the ../2 macro, auto-imported from Kernel:

    iex> range = 1..3
1..3
iex> first .. last = range
iex> first
1
iex> last
3
